A program written in C++ to solve the following psuedo code problem:

set count1 = 0
set count2 = 0
read number
while read was successful do
if number mod 2 equals 0
then
set count1 = count1 + 1
otherwise
if number div 7 equals 0
then
set count2 = count2 + 1
read number
write count1
write count2
